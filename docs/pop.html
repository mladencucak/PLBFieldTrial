Import the data and set the levels of the treatment (3 fixed and 3 variable doses) and variety (susceptible to resistant).

  group_by(date) %>% 
  count(Genotype) %>%
  rename(counts = n) %>% 
  mutate(prop = prop.table(counts)) %>%
